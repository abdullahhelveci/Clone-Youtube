# CLONE-YOUTUBE 

This project is a YouTube clone. Users can filter videos by categories. They can click on the video they want to watch, view video details, and check out recommended videos.

## Used Technologies

- Javascript
- React
- Context API
- Tailwind
- React Icons
- Axios
- Millify
- React-player
- React-router-dom
